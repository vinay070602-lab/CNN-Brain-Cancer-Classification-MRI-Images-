# CNN-Brain-Cancer-Classification-MRI-Images-
Developed a deep learning model using Convolutional Neural Networks (CNN) to classify brain tumors from MRI images into different tumor categories. The project involved image preprocessing, model training, and evaluation using TensorFlow/Keras.


## Project Overview
The objective of this project is to apply deep learning techniques to medical imaging for accurate brain tumor classification.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- OpenCV

## Steps Performed
1. MRI image preprocessing
2. Dataset preparation
3. CNN model architecture
4. Model training
5. Model evaluation

## Deep Learning Model
- Convolutional Neural Network (CNN)
